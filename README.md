 ## 使用

- 打开：右上角文件->打开->LawnStrings.json 或 ZombieString.json
- 写入：右上角文件->保存 或 另存为
- 编辑：
    - 新建：在当前分类下的最后一位Id新建一个对象
    - 应用：将当前改动同步到该对象（不是保存！）
    - 删除：删除该对象
    - 重置：将该对象重置到上一次保存

> 注意！编辑时的改动只有在保存后才会写入json文件！
